Typography Control

Typography works with a golden ratio scale. Their default bootstrap 3 margin can be overriden with the class or Sass include normalize-text on any parent element.

Additionaly, all B3 typography has been extended to work with rem units which dynamically adapt to screen sizes.

To get full control of the text re-sizing on different devices head up to the _text-responsive.scss file and modify the ammount of units that typography needs to be resized relative to the html element $font-size-base variable.

Buttons

EBM extends B3 buttons by creating outlined buttons based on the brand, grayscale, blue and social colors' list.
Use the class syntax: .btn-outline .btn-#{color-sufix} to use these buttons directly in your html or check out the _buttons-custom.scss file.

One important thing about the EBM .btn extension is that buttons either on anchors or button elements are not affected by parent .text-color classes

Grid lists

Grid lists are initially taken from zurb foundation's grid-blocks and adapted to B3

NOTE: This image list created with the .grid-list .grid-list-#{num-of-columns}-#{breakpoint} class has also images being dynamically processed by the Grunt responsive_images task which generates images for different screen sizes to increase REST performance.

Additionaly, add the isotope-grid class that, if included in the src/coffee, folder, it will extend grid-lists with the metafizzy's isotope jQuery plugin.

Banners

Banners are modular presentational blocks that can be used both vertically or horizontally. They can be used with B3 grid-system to manipulate their sizes and breakpoints.

One of the main benefits of using EBM banners is that no matter how big images are inside the banner-pic they are set to fit the available space.

Control the ebm-banners sizes as well as other elements or components within the _global.scss file.

Container sizes

The B3 $container-large-desktop has been set to $container-desktop to have more control over the perceived width with retina display screens. If resized, these containers will fit the screen accordingly.

The text elements that are being centered in the containers are possible with the .flex-middle-center EBM class and mixin in combination with a .minh-#{$ebm-times-unit*10}. Know more about the EBM flexbox mixins and classes here.
